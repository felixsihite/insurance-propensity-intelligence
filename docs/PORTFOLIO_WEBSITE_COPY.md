# Portfolio Website Copy

## Card Title

Insurance Customer Propensity Prediction

## Short Description

End-to-end Data Science project that ranks health insurance customers by vehicle insurance cross-sell propensity using leakage-safe feature engineering, lift analysis, customer segmentation, and a Streamlit scoring dashboard.

## Highlight Bullets

- Ranked 127,037 test customers with a reusable propensity scoring pipeline.
- Achieved ROC-AUC 0.8585 and PR-AUC 0.3685 on stratified validation.
- Captured 79.40% of validation responders in the top 30% ranked population.
- Delivered SQL reporting, model interpretability, dashboard screenshots, and portfolio-ready light/dark visual assets.
